rundeck-opmanager

A hack to include nodes from OpManager in rundeck

Download this project as a .zip file Download this project as a tar.gz file

Autogenerate nodes list fom OpManager in RunDeck

This automatic nodelist generator is a small hack I've made to populate RunDeck projects with nodes from OpManager. It's very very ugly at the moment, so I'd gladly receive some pull requests to make this a little bit less ugly.

Create an API Key

To access your nodes you need an API key in OpManager. You can read a little bit more about that here.
When this is done, edit the URL in getnodes.rb, and put both files in a folder named bin in the rundeck users home folder.

Configure RunDeck project

Now it's time to create the project. Log on to RunDeck and create a new project. Click "Add Source" then "script". Resource format is resourceyaml and "Script File Path" is /var/lib/rundeck/bin/genYaml.sh in my case.
